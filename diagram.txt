OUTPUTS AFTER main.py
=====================

1. data/generated/spy_open_setup_raw.csv
- 5 SPY bars per day
- 09:30 through 09:34 only
- direct LLM query input

2. data/generated/spy_open_setup_features.csv
- one row per day
- engineered setup summary
- VIX context
- outcome-window labels
- ground-truth answer key


TRAIN / TEST SPLIT
==================

Training corpus for retrieval:
- 2010-01-04 through 2025-03-01

Testing / evaluation period:
- 2025-03-02 through present

Rule:
- test-period rows are never inserted into Chroma

KEY IDEA
========

The LLM never sees the future path for the current test day.

It only sees:
- the current day’s 09:30 to 09:34 setup bars
- rules
- historical training examples from older dates

The outcome window is used only to build the answer key.
